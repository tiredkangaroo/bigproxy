cap

fast and configurable proxy with MITM

Install

Clone the repository

git clone https://github.com/tiredkangaroo/bigproxy.git

Change the working directory to the Git Repository folder

Install dependencies (you must have golang and node + npm) 

go mod tidy && npm i --prefix manager

Set the Environment Variables

PROXY_CACERT

Relative (from the running directory) or absolute path to the PEM-encoded X509 CA certificate file. 

PROXY_CAKEY

Relative (from the running directory) or absolute path to the PEM-encoded PKCS8 CA key file. 

PROXY_CONFIG_FILE

Relative (from the running directory) or absolute path to the save location for the proxy configuration file. The program must be allowed r/w access.

Run the project

go run make.go run
